# STAPLE 2020 README

This data contains the non-blind release of all splits of the data from STAPLE: Simultaneous Translation
and Paraphrasing for Language Education, a shared task held at the Workshop on Neural Generation and
Translation (WNGT) in 2020.

## Data

There are five folders, one for each language. Each folder contains six files:

1. Train split: shared task data and AWS baseline
2. Dev split: shared task data and AWS baseline
3. Test split: shared task data and AWS baseline

## Code

For code related to the shared task, please see: [https://github.com/duolingo/duolingo-sharedtask-2020]()
